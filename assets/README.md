## Suggested runtime sizing

- Building icons: render around `64x64` to `96x96`.
- Resource/category/research icons: render around `24x24` to `48x48`.
- Unit portraits/sprites: render around `32x32` to `80x80`.
- Animated unit sprites: use walk cycle order `1 -> 2 -> 3 -> 2`.
- `tiles/ground.png`: use repeating texture mode.
- `tiles/grid.png`: optional overlay for placement/debug visuals.
